# Gym Roleplay Training

A Next.js prototype for gym coaches who need to practice member conversations against real program context.

The first screen is the roleplay lab: pick a scenario, read the member's opening line, write the coach response, score it against the checklist, and keep the linked workout template visible below.

## Run

```bash
bun install
bun run dev
```

Open `http://localhost:3000`.

## Verify

```bash
bun run lint
bun run build
```

## Product Surface

- Roleplay scenarios for new-member nerves, plateau frustration, and adherence resets.
- Typed coaching rubrics and per-scenario success checklists.
- Program folders, editable workout templates, and local revision storage.
- Strong-style history reference, PR graph, and compiled phone/watch session packet.
